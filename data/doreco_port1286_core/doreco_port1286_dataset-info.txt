The Daakie DoReCo dataset was compiled by Manfred Krifka between 2010 and 2018 and further processed by the DoReCo team (in particular Runzhi Lou, Xia Qiang, Ludger Paschen, and Matthew Stave) between 2019 and 2022. The files that the Daakie DoReCo dataset are based on are part of a larger collection of Manfred Krifka's Daakie data.

Daakie DoReCo annotation files are distributed under an Attribution 4.0 International (CC BY 4.0) License (https://creativecommons.org/licenses/by/4.0/).

Daakie DoReCo audio files are distributed under an Attribution 4.0 International (CC BY 4.0) License (https://creativecommons.org/licenses/by/4.0/).

The Daakie DoReCo dataset should be cited as follows:
Krifka, Manfred. 2022. Daakie DoReCo dataset. In Seifart, Frank, Ludger Paschen and Matthew Stave (eds.). Language Documentation Reference Corpus (DoReCo) 1.2. Please note that when citing this dataset, or any number of DoReCo datasets, it is NOT sufficient to refer to DoReCo as a whole, but the full citation for each individual dataset must be provided, including the name(s) of the creator(s) of each data set.

A web interface for easy access to this dataset as well as other DoReCo datasets and further information can be found at https://doreco.huma-num.fr.
